# Customer Churn Prediction using Artificial Neural Networks (ANN)

## Overview

This project predicts whether a bank customer will leave the bank (churn) using a deep learning model based on an Artificial Neural Network (ANN). The model analyzes customer demographic and financial data to identify patterns that indicate potential churn.

## Technologies Used

* Python
* TensorFlow
* Keras
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset

The project uses the **Churn_Modelling.csv** dataset, which contains customer information such as:

* Credit Score
* Geography
* Gender
* Age
* Tenure
* Balance
* Number of Products
* Has Credit Card
* Active Member Status
* Estimated Salary

## Project Workflow

1. Data preprocessing
2. Handling categorical variables (encoding)
3. Feature scaling
4. Train-test split
5. Building the Artificial Neural Network (ANN) model
6. Applying Early Stopping
7. Model training
8. Model evaluation and visualization

## Model Evaluation

The model performance is evaluated using:

* Confusion Matrix
* Accuracy Score
* Training vs Validation Loss

## Results

The ANN model successfully learns patterns from the data and predicts customer churn with good accuracy. Visualizations such as loss curves and confusion matrices help analyze model performance.

## How to Run the Project

1. Clone the repository
2. Install required libraries using pip
3. Open the notebook and run all cells
